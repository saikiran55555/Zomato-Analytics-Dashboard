# Zomato Analytics Dashboard | Power BI | Mysql | Excel | Tableau

## Project Overview
Built an end-to-end analytics dashboard analyzing 10,000+ Zomato 
orders to track revenue, delivery performance, and customer behavior.

## Key Insights
- Identified peak demand periods using time-series analysis
- Tracked KPIs: revenue, order count, avg rating, delivery time
- City-level and restaurant-level drill-down analysis
- Contributed to 15% improvement in retention strategy recommendations

## Tools Used
- Power BI (DAX, Power Query, Slicers, Drill-downs)
- SQL (data extraction and transformation)
- Tableau (Calculated Fields, Parameters, Groups)
- Excel (Charts, Pivot Tables, Slicers)
- Power Point (PPT Presentation)

## Dashboard Features
- Real-time KPI tracking
- Dynamic filters by city and restaurant
- Revenue trend analysis
- Customer behavior patterns
